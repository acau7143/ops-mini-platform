## 오늘 한 일

## 오늘 배운 점
- docker logs는 컨테이너가 없으면 바로 실패한다
- set -euo pipefail 환경에서는 || echo "[WARN]" 처리가 필수
- Docker 컨테이너 안 Nginx는 외부 요청을 172.17.0.1로 받는다
  → Docker 네트워크 게이트웨이를 거쳐서 들어오기 때문
  → 실제 외부 IP는 로그 맨 뒤에 따로 기록됨
- libredtail-http 같은 자동 스캐너가 PHPUnit 취약점 경로를
  주기적으로 스캔하고 있었음 → 전부 404로 막힘
